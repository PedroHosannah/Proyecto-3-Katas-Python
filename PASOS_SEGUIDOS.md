# Pasos seguidos durante el proyecto

Primero leí los enunciados y separé los ejercicios por número. Después resolví cada uno usando estructuras sencillas y las funciones indicadas en cada ejercicio.

He usado `map()`, `filter()`, `reduce()` y funciones lambda cuando se pedían expresamente. En el resto he utilizado principalmente variables, listas, bucles y condicionales para que el código sea fácil de seguir.

También añadí controles de errores en los ejercicios que lo necesitaban y comentarios breves para explicar las partes principales. Finalmente ejecuté el archivo completo para comprobar su funcionamiento.
