# Proyecto 3 - Katas Python

## Autor
Pedro Hosannah Navas

## Descripción
Este repositorio contiene los 40 ejercicios del Proyecto 3 de Python. Los ejercicios están resueltos con código sencillo, funcional y ordenado.

Cada ejercicio aparece encabezado por su enunciado y contiene comentarios breves para explicar las decisiones principales.

## Contenidos trabajados
- Tipos de datos y estructuras básicas.
- Condicionales y bucles.
- Funciones y funciones lambda.
- `map()`, `filter()` y `reduce()`.
- Recursividad.
- Manejo de excepciones.
- Clases y objetos.
- Uso de módulos.

## Cómo ejecutar el proyecto

```bash
python katas_python.py
```

El bloque final contiene pruebas de funcionamiento. Las llamadas que solicitan datos por teclado están comentadas para que el archivo pueda ejecutarse completo sin detenerse.

## Archivos
- `katas_python.py`: contiene los 40 ejercicios.
- `PASOS_SEGUIDOS.md`: explica brevemente el proceso.
- `.gitignore`: evita subir archivos innecesarios.
